# 🐍 Python Practice Workspace

Welcome to the **Python Practice** repository! This is a curated collection of scripts, exercises, and mini-projects designed to master Python from the ground up—covering everything from basic syntax to object-oriented programming and data analysis.

---

## 📂 Project Structure

The repository is organized into thematic files and sub-projects:

### 🛠️ Fundamentals & Core Concepts
- **`index.py`**: Introduction to variables, types, and basic syntax.
- **`conditionals.py` & `loops.py`**: Mastery of control flow and iteration.
- **`strings.py`**: Deep dive into string manipulation and formatting.
- **`functions.py`**: Practice with reusable code blocks and scoping.

### 📊 Data Structures
- **`listsAndTuples.py`**: Working with ordered sequences.
- **`dictAndSets.py` & `sets.py`**: Understanding key-value pairs and unique collections.

### 🏗️ Advanced Topics
- **`oops.py` & `oops2.py`**: Object-Oriented Programming (Classes, Constructors, Static Methods, and Decorators).
- **`filesIo.py` & `filesIo2.py`**: Handling file read/write operations.
- **`temp.py`**: A sandbox for quick experimentation and testing.

### 🎮 Games & Projects
- **`game.py` & `game2.py`**: Logic-based mini-games (e.g., Guess the Number).
- **`python-for-ai/`**: A specialized directory for AI-related Python practice.
  - **`sales-analysis/`**: A practical data analysis project that processes sales data from CSV, JSON, and Excel formats.

---

## 🚀 Key Features

- **Hands-on Examples**: Each script is self-contained and focuses on a specific Python feature.
- **Real-world Practice**: Includes a multi-format sales analyzer to simulate data engineering tasks.

## 🛠️ Getting Started

### Prerequisites
- Python 3.x
- pip (for external libraries)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Python-Practice.git
   ```
2. (Optional) Install dependencies for the sales analysis project:
   ```bash
   pip install -r python-for-ai/requirements.txt
   ```

### Running a Script
Simply execute any file using the Python interpreter:
```bash
python oops.py
```

## 📈 Learning Path
1. Start with `index.py` for basics.
2. Move to `conditionals.py` and `loops.py`.
3. Explore Data Structures (`listsAndTuples.py`, `dictAndSets.py`).
4. Master OOP concepts in `oops.py`.
